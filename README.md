## 📊 Proyectos destacados

### 🚗 Análisis de Accidentalidad Vial en Antioquia (2014–2022)
Dashboard interactivo de 4 páginas construido en Power BI con datos reales del gobierno colombiano.

**Hallazgos clave:**
- El 76% de los accidentes son choques
- Las horas más peligrosas son entre las 14:00 y las 18:00
- 2014 fue el año más crítico con 167 accidentes registrados
- El 90% ocurre con clima normal — el factor humano es determinante

**Stack:** Power BI · Power Query · DAX · OData · GitHub

🔗 [Ver proyecto](https://github.com/camiloroman-dataanalyst/accidentalidad-vial-antioquia)
